# Lead Manager Automation

## Overview
This repository contains automation for Lead Manager web application:
- UI Automation using Selenium WebDriver + TestNG + Maven
- API Testing scenarios for Login and Leads APIs 
- Manual test cases created in excel >  addin folder lead-manager-automation\TestCases (Osmos_Manual_API_Assignment.xlsx)
---

## Tools and Frameworks Used
- Java 17+
- Selenium WebDriver 4.18
- TestNG 7.9
- Maven 3.x
- WebDriverManager 5.7
- Postman / Rest Assured for API testing
- Chrome / Firefox / Edge browser
